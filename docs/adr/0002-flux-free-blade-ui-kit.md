# Flux Free as Blade UI kit

Authenticated app UI (Login gate, shell, Today, Monthly, expenses, profile) will use Livewire Flux Free as the Blade component kit, themed to the ledger ink system in ADR 0001—not Flux’s stock look. Routes stay controller → Blade with classic forms; Livewire is installed mainly as Flux’s runtime. Flux Pro is deferred until a concrete Free gap appears. We rejected a Livewire/Volt rewrite, adopting Flux’s default aesthetic, and leaving Breeze UI as a permanent dual stack.
